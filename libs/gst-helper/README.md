# gst-helper

Gstreamer helper project

## Dependencies:

Projects referencing this project must have following cmake targets defined:

* fmt::fmt
* tcam::tcamprop
* outcome::outcome
* dutils_img::base


## Libraries exported:

* tcam::gst-helper
* tcam::gst-helper-dutils
* tcamprop_system::base
* tcamprop_system::provider
* tcamprop_system::consumer
